## 📖 Описание программы

**Weather Diary** — это графическое приложение для ведения дневника погоды. Программа позволяет записывать ежедневные наблюдения за погодой, фильтровать записи и анализировать статистику.

### ✨ Основные возможности:

- ✅ **Добавление записей** с указанием даты, температуры, описания и наличия осадков
- ✅ **Валидация ввода** (проверка формата даты, температуры, заполнение полей)
- ✅ **Фильтрация записей**:
  - По дате (выбор конкретного дня)
  - По температуре (показать записи с температурой выше заданного значения)
  - По осадкам (все / с осадками / без осадков)
- ✅ **Сохранение и загрузка** данных в JSON файл
- ✅ **Статистика** (общее количество записей, средняя/макс/мин температура, количество дождливых дней)
- ✅ **Примеры записей** при первом запуске
- ✅ **Удобный табличный интерфейс** с прокруткой
